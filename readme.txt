== Description ==

This plugin makes it easy to add draugiem.lv/lapas (frype.com) fan page to sidebar of WordPress blog. ( http://mediabox.lv/blog/wordpress/spraudni/draugiem-lv-lapas-fanu-lapa/ )

You `MUST` have a page on draugiem.lv/lapas to use this widget.

Lai lietotu spraudni ir jābūt izveidotai savai lapai draugiem.lv 

= Features =

* Custom CSS. ''relative'' and ''absolute'' URIs are supported. (Since 0.1.3)
* '''In Your language'''. Defaults: '''English''' and '''Latvian''' (Since 0.1.2)
* Widget title
* Fan count, images and URLs
* Page description (draugiem.lv/lapas)

== Installation ==

Installation:

* Download the plugin.
* Copy the `draugiemlvlapas-fan-page` (or `frypepage_widget` in older versions (tests)) folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* An options panel will appear under Widgets.
* Add fanpage URL and enjoy!

= Fan page URL =
* Fan page URL  is part of http://draugiem.lv/umbrovskiscom You will need to insert only `umbrovskiscom`
* Instructions: [Draugiem.lv/lapas Fan page](http://mediabox.lv/en/wordpress-plugins/draugiem-lv-lapas-fan-page/)
